# Build Resources

## Icon

Поместите файл `icon.ico` (256x256px) в эту папку для сборки Windows installer.

### Инструменты для создания:

- **Online:** https://www.icoconverter.com/
- **GIMP:** Бесплатный редактор изображений
- **Photoshop:** Профессиональный редактор

### Требования:

- Формат: .ico
- Размер: 256x256 пикселей (рекомендуется)
- Прозрачный фон: опционально

### Советы:

1. Используйте простой дизайн
2. Избегайте мелких деталей
3. Тестируйте на разных размерах
4. Используйте контрастные цвета
